# 鼠标移动模拟器 (Raspberry Pi Pico)

一个基于 Raspberry Pi Pico 的自然鼠标移动模拟器，旨在模拟真实人类的鼠标操作行为，产生更加自然、真实的鼠标移动轨迹。

## 硬件要求

- **Raspberry Pi Pico** (RP2040)
- **可选**: WS2812B 彩色 LED (连接至 GP16 引脚)，用于指示当前工作模式

## 主要功能

- 模拟自然的人类鼠标移动行为
- 支持多种移动模式
- 非匀速运动（模拟真实的手部加速/减速运动）
- 实时 LED 状态指示

## 核心特性

- **自然移动**: 模拟人类鼠标移动的非匀速特性（加速-匀速-减速模式）
- **多样化行为**: 包含 6 种不同的移动模式
- **智能暂停**: 模拟人类思考和观察的停顿行为
- **视觉反馈**: 通过 LED 提供实时模式指示
- **低功耗**: 专为 Pi Pico 的低功耗特性优化

## 移动模式

1. **网页浏览模式** 🟢: 模拟用户在网页上浏览的行为，包括小范围的局部移动和停留观察
2. **页面扫描模式** 🔵: 模拟从左到右的页面扫描行为，模拟人类阅读时的视线移动
3. **探索性移动模式** 🟡: 模拟用户在页面上随机探索的行为，包含小幅跳转和观察
4. **随机移动模式** 🟣: 模拟用户随意移动鼠标的行为，增加活动的自然性
5. **圆形移动模式** 🟠: 鼠标沿圆形轨迹移动，模拟轻微的手部微震和调整
6. **目标聚焦模式** 🔴: 模拟用户专注于特定区域的行为，包括微调和小范围移动

## 快速开始

### 硬件连接

1. 将 Raspberry Pi Pico 通过 USB 连接到目标计算机
2. (可选) 将 WS2812B LED 连接到 GP16 引脚，GND 接地，VCC 接 3.3V

### 软件部署

1. 将 `code.py` 文件复制到 Raspberry Pi Pico 的文件系统中
2. 确保 `lib` 目录下包含必要的库文件 (`adafruit_hid` 和 `neopixel.mpy`)
3. 重新启动 Pi Pico

### 运行

设备将自动以 USB HID 鼠标身份连接到计算机，并立即开始模拟鼠标移动行为。

| ------- | -------- | ---------------------- |
| 🟢 绿色 | 网页浏览 | 模拟网页浏览行为       |
| 🔵 蓝色 | 页面扫描 | 模拟页面扫描/阅读行为  |
| 🟡 青色 | 探索移动 | 模拟页面探索行为       |
| 🟣 紫色 | 随机移动 | 模拟随机鼠标移动       |
| 🟠 橙色 | 圆形移动 | 模拟圆形轨迹移动       |
| 🔴 红色 | 目标聚焦 | 模拟专注区域的微调移动 |

## 自定义配置

您可以根据需求自定义以下参数：

- **移动速度**: 修改代码中的移动步长和时间间隔
- **模式时长**: 调整每个模式的持续时间和切换频率
- **LED 设置**: 更改 LED 颜色或禁用 LED 功能
- **移动范围**: 调整鼠标移动的幅度和范围
- **行为模式**: 添加或修改移动模式算法

## 技术细节

本项目利用了人类运动学原理，通过编程实现：

- 非线性移动轨迹（加速/减速曲线）
- 随机微调和停顿
- 不规则的移动速度变化
- 符合人体工程学的移动模式
