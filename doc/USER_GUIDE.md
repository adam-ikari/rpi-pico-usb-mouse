## 运行说明

### 启动行为

设备接入计算机后：

1. **自动识别为 USB 鼠标**：无需安装驱动
2. **立即开始移动**：约 1-2 秒后开始移动
3. **模式自动切换**：每 15-120 秒自动切换模式
4. **LED 指示当前模式**：不同颜色对应不同模式

### 停止方法

- 直接拔出 USB 线即可停止

### 工作状态

设备工作时：

- **LED 呼吸灯**：停顿时亮度缓慢变化
- **LED 恒定亮度**：移动中保持最大亮度
- **颜色过渡**：模式切换时平滑过渡 2 秒

## LED 指示说明

### 颜色含义

| 颜色    | 模式名称 | 行为描述                         | 典型场景           |
| ------- | -------- | -------------------------------- | ------------------ |
| 🔴 红色 | 网页浏览 | 中等范围局部移动，3-8 次小幅调整 | 浏览网页、查看文档 |
| 🟢 绿色 | 页面扫描 | 横向来回扫描，向右慢、向左快     | 阅读长文、扫描表格 |
| 🔵 蓝色 | 探索移动 | 大范围跳跃式探索，2-4 次         | 查找内容、浏览图库 |
| 🟡 黄色 | 随机移动 | 随意移动，2-5 次，±80px          | 随意浏览、活动检测 |
| 🟣 紫色 | 圆形移动 | 大半径椭圆轨迹，慢速旋转         | 审视区域、等待加载 |
| ⚪ 青色 | 目标聚焦 | 小范围频繁微调，最多 5 次        | 精确操作、细节查看 |

### 正常工作状态

#### 移动中

- **亮度**：恒定最大亮度 100%
- **颜色**：对应当前模式
- **变化**：位置实时变化，无闪烁

#### 停顿中

- **亮度**：呼吸灯效果 10%-100%
- **周期**：约 2-3 秒一个周期
- **颜色**：对应当前模式

#### 模式切换

- **颜色过渡**：2 秒平滑过渡到新颜色
- **亮度变化**：从停顿的呼吸灯渐变到活动的恒定亮度
- **无闪烁**：使用 ease-in-out 缓动函数

## 故障排除

### 快速红色闪烁（0.1 秒间隔）

- **原因**：程序严重错误或崩溃

### 慢速红色闪烁（0.2 秒间隔）

- **原因**：USB 设备错误或 I/O 错误
- **处理**：更换 USB 线或尝试其他 USB 端口

### LED 完全不亮

- 更换 USB 数据线
- 尝试其他 USB 接口

### 设备无法识别

**症状**：插入后计算机无反应

**解决方法**：

- 更换 USB 数据线
- 尝试其他 USB 接口

### 鼠标不移动

**症状**：设备已连接但鼠标不移动

**解决方法**：

- 在其他计算机上测试

### 鼠标动得很卡

**怎么办**：

- 换个 USB 口试试

### 被安全软件拦截

**症状**：设备被安全软件标记或拦截

**解决方法**：

- 联系 IT 部门申请使用许可
- 在个人计算机上使用

⚠️ **提醒**：企业环境使用前需获得 IT 部门批准。

## 常见问题

### Q1: 设备是否会被检测为可疑设备？

**A**：本设备显示为标准 USB HID 鼠标，与普通鼠标无异。但部分企业安全软件可能会监控外接 USB 设备。

---

### Q2: 可以连接多个设备吗？

**A**：可以。每个设备都会被识别为独立的 USB 鼠标。但建议不要超过 2 个，以免移动过于复杂。

---

### Q3: 支持哪些操作系统？

**A**：支持所有支持 USB HID 鼠标的操作系统：

- ✅ Windows 7/8/10/11
- ✅ macOS 10.12+
- ✅ Linux（所有发行版）
- ✅ Chrome OS
- ✅ Android（需 OTG 支持）

---

### Q4: 电源需求是多少？

**A**：

- 任何 USB 端口均可供电（USB 2.0 标准为 500mA）
